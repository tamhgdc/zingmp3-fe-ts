# Zing Mp3 Clone (Vue 3)
Vue 3, Vite, TypeScript, SCSS

## Installation:
```sh
git clone https://github.com/uchindev/zingmp3-fe-ts.git
cd zingmp3-fe-ts
npm install
npm run dev
```
